# **App Name**: CYBA Galaxy

## Core Features:

- Home Page: Landing page with a brief overview of CYBA and its services.
- Membership Page: Details about CYBA's membership options and benefits.
- Merch Store: Online store for purchasing CYBA merchandise.
- Blog: A space for CYBA to share industry insights and news.
- Cybaboard: A community forum for members to interact.

## Style Guidelines:

- Primary color: Violet (#8A2BE2) for a futuristic, music-oriented feel.
- Background color: Dark gray (#222222) for a retro aesthetic.
- Accent color: Light blue (#ADD8E6) for contrast and a vintage vibe.
- Font: 'Space Grotesk' sans-serif, for both headlines and body, for a computerized feel.
- Neon-style icons related to music and technology.
- Retro-inspired layouts with geometric shapes and a clear content hierarchy.
- Subtle animations to mimic a VHS tape effect.
